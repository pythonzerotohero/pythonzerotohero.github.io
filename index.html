Introduction:

This course is for those students who have zero experience in coding. We will use python language, which is easy for noobs. We will cover a lot of topics during one month. Yeah, this course will help you become a master of python in one month. Topics we cover: python basics, using python to manipulate data, using python to draw some statistical charts, and python to build games. There is one project, four homeworks, and one final exam.
